# ProductLens Analytics Dashboard

A modern data-driven product analytics platform built to help product teams monitor user engagement, retention, conversion, and feature adoption through a clean interactive dashboard.

## Features
- Executive KPI overview
- Product usage analytics
- Retention and churn tracking
- Funnel visualization
- Feature adoption insights
- Responsive SaaS-style dashboard UI

## Tech Stack
- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- Vitest

## Getting Started
```bash
npm install
npm run dev
